# Decision Record

## Decision
Adopt a structured continuity and logging protocol centered on:
- `plans/current.md`
- `plans/backlog.md`
- `logs/sessions/YYYY-MM-DD.md`
- `decisions/*.md`
- `PLAN.md`

## Context
Work sessions need to resume cleanly without depending on prior chat context. The project requires a low-noise operational handoff system.

## Alternatives considered
- Continue using ad hoc chat-history memory only.
- Keep only one large running notes file.
- Store continuity solely in task-specific plan files.

## Status
Accepted
