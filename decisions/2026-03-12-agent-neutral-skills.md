# Decision Record

## Decision
Standardize local skill documentation and guidance to be agent-neutral.

## Context
Project instructions and continuity rules are intended to be reusable across different agent runtimes. Vendor-specific phrasing in skills created avoidable coupling and confusion.

## Alternatives considered
- Keep existing vendor-specific wording in skill docs.
- Convert only top-level project docs and leave skill docs unchanged.
- Fully standardize all local skill docs while keeping runtime-specific CLI details only where technically required.

## Status
Accepted
