# Docs

## Purpose
This folder stores durable project documentation that should be easy to discover without scanning the full repository.

## Current docs
- `continuity-protocol.md` — operational continuity/logging protocol summary and file map.

## Notes
- Active execution state always lives in `plans/current.md`.
- Queued work always lives in `plans/backlog.md`.
- Session summaries live in `logs/sessions/YYYY-MM-DD.md`.
