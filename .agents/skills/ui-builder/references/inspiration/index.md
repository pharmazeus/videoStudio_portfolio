# Inspiration Index

This file documents every image in the `inspiration/` folder.
Before adding an image, add an entry here: what to extract, what to ignore.
The agent reads this file before looking at any inspiration image.

---

## How inspiration images work

Inspiration images are NOT targets — they are structural references.
Extract: layout logic, spacing rhythm, hierarchy pattern, interaction idea.
Discard: colors, fonts, brand identity, anything visually specific to their product.
Apply: project design tokens to implement what you extracted.

---

## Template (copy for each new image)

```
## [descriptive-filename].png
Source: [URL or "provided by user on DATE"]
What this is: [brief description of what's shown]

Extract:
- [structural/layout idea worth borrowing]
- [spacing or hierarchy pattern]
- [interaction pattern]

Ignore:
- Their color palette
- Their font choices
- [any other specific visual identity to discard]

Notes:
- [anything else the agent should know before using this image]
```
