# Asset Spec

## Purpose
This reference defines the output format for website preview media generated by the `video-preview-prep` skill.

## Preview clip presets
### Portrait
- Dimensions: `720x1280`
- Codec: `H.264`
- Container: `mp4`
- FPS: `30`
- Duration: `5` seconds default
- Audio: muted

### Landscape
- Dimensions: `1280x720`
- Codec: `H.264`
- Container: `mp4`
- FPS: `30`
- Duration: `5` seconds default
- Audio: muted

## Poster preset
- Format: `jpg`
- Timestamp: `1.0s` default
- Quality: web-optimized

## Naming convention
- Preview: `<slug>-preview.mp4`
- Poster: `<slug>-poster.jpg`

## Output folders
- `public/videos/previews/portrait/`
- `public/videos/previews/landscape/`
- `public/posters/portrait/`
- `public/posters/landscape/`

## Manifest shape
```json
{
  "generatedAt": "2026-03-13T00:00:00.000Z",
  "items": [
    {
      "slug": "example-slug",
      "sourcePath": "raw/example.mov",
      "orientation": "portrait",
      "previewSrc": "/videos/previews/portrait/example-slug-preview.mp4",
      "poster": "/posters/portrait/example-slug-poster.jpg",
      "duration": 5,
      "width": 720,
      "height": 1280
    }
  ]
}
```

## Integration note
The manifest is operational only. In v1, update `caseStudies` manually or semi-manually by copying generated paths into each `media` object.
